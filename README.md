\# AI-Powered Waste Generation Prediction System



This project predicts waste generation patterns using machine learning and provides insights through a web dashboard.



\## Tech Stack

\- Python

\- FastAPI

\- Machine Learning

\- React

\- Data Visualization



\## Features

\- Predict waste generation by location and time

\- Interactive dashboard for visualization

\- AI-generated insights for waste management decisions



\## Project Structure

backend/ – ML model and API  

frontend/ – dashboard UI  



\## Dataset

Ward-level waste data from Mysuru city.

\- Real-time data integration
